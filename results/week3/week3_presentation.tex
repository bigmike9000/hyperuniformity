\smallskip
\textbf{1.\ Dense Bragg spectrum.}\\
BT has ${\sim}10^5$ peaks in $k<50$
($24\times$ more than Fibonacci).
$\mathcal{S}(t)$ oscillates log-periodically
with period $2\ln\theta_1\!\approx\!1.62$,
masking power-law decay.

\medskip
\textbf{2.\ Large sub-leading eigenvalue.}\\
$|\lambda_2|=0.802$ (BT) vs.\ $0.618$ (Fib).
Even fitting over $t\in[10^5,10^9]$
gives $\hat\alpha=1.67$, not 1.545.

\medskip
\colorbox{resultbg}{\parbox{0.94\linewidth}{\smallskip
  \textbf{Solution:} $\bar\Lambda$ from $\sigma^2(R)$
  averages over oscillations.\\
  $\bar\Lambda_{\rm BT}=0.377$, stable from $N\!\sim\!2000$.
\smallskip}}
\end{column}
\end{columns}
\end{frame}
\note{
Rod overlap: minimum BT spacing $=1.000$; needed $2a<1.000$;
corrected rod radius to $a=0.45$.
Variance-based $\bar\Lambda$ is robust because the running average over $R$
cancels log-periodic oscillations.
Conclusion: use $\bar\Lambda$ (not spreadability) for any chain with large $|\lambda_2|$.
}

\begin{frame}{How well do our estimates converge with $N$?}

\begin{columns}[T]
\begin{column}{0.56\textwidth}
\centering
\includegraphics[width=\linewidth]{figures/fig_finite_size_alpha.png}
\end{column}
\begin{column}{0.42\textwidth}
\smallskip
\textbf{(a) $\bar\Lambda$ from $\sigma^2(R)$:} rock-stable.\\[2pt]
BT and Fibonacci both converge by $N\!\sim\!10^3$.
Spatial averaging cancels oscillations.

\medskip
\textbf{(b) $\hat\alpha$ from $S(k)\!\sim\!k^\alpha$:} fails.\\[2pt]
$S(k)$ is pure Bragg peaks ---
no smooth $k^\alpha$ background to fit.
\begin{itemize}\setlength\itemsep{2pt}
  \item BT: $\hat\alpha$ bounces $0.9$--$2.1$ (true $1.545$)
  \item Fib: $\hat\alpha$ bounces $1.8$--$2.7$ (true $3$)
  \item 0222: gives $\hat\alpha<0$ (true $0.639$)
\end{itemize}

\medskip
\colorbox{resultbg}{\parbox{0.94\linewidth}{\smallskip
  \textbf{Conclusion:} always use $\bar\Lambda$ (or $C_{II}$, $A_{III}$)
  from $\sigma^2(R)$; never fit $S(k)$ directly.
\smallskip}}
\end{column}
\end{columns}
\end{frame}
\note{
$S(k)$ for a substitution tiling is a superposition of delta functions; no smooth
$k^\alpha$ background to fit.
0222 gives \emph{negative} $\hat\alpha$ because the Bragg peaks at small $k$
decrease with $k$ (spectral weight concentrates near $k=0$).
Class~I $\bar\Lambda$ is a spatial average; fluctuations cancel when averaged,
giving stable convergence from $N\sim1000$.
}

\begin{frame}{Searching for a $2<\alpha<3$ quasicrystal}

Enumerated all $3\times3$ unimodular Pisot matrices with row sums $\leq4$:
\[
  39{,}304\ \xrightarrow{\text{Pisot}}\ 4{,}882
  \ \xrightarrow{\text{candidate }\alpha\in(2,3)}\ 305
\]
\colorbox{newbg}{\parbox{0.97\linewidth}{%
  All 305 give $\alpha=2$ or $\alpha=3$ exactly. \textbf{The gap is empty.}}}

\medskip
This is not a coincidence.
For any unimodular matrix with a complex conjugate sub-leading eigenvalue pair:
\begin{gather*}
  \underbrace{|\det M|}_{=1} = \lambda_1|\lambda_2|^2
  \;\Rightarrow\; |\lambda_2| = \tfrac{1}{\sqrt{\lambda_1}} \\[0.3em]
  \Rightarrow\; \alpha = 1-\frac{2\ln|\lambda_2|}{\ln\lambda_1} = 2
  \quad\text{(exactly, for any }\lambda_1\text{)}
\end{gather*}
For all-real sub-leading eigenvalues: exhaustive search finds only $\alpha=3$.
\end{frame}
\note{
Pisot condition: all sub-leading eigenvalues have $|\lambda_i|<1$ (pure Bragg
diffraction, no diffuse component).
Complex-pair identity: $\det M=\lambda_1|\lambda_2|^2$. For $\det M=1$:
$|\lambda_2|=1/\sqrt{\lambda_1}$ exactly, pinning $\alpha=2$ regardless of $\lambda_1$.
All-real case confirmed exhaustively; no analytical proof yet.
}

\begin{frame}[shrink=2]{Metrics for Classes II and III}

\begin{columns}[T]
\begin{column}{0.58\textwidth}
\centering
\includegraphics[width=\linewidth]{figures/fig_generalized_metrics.png}

\vspace{0.2em}
{\footnotesize Each normalized variance plateaus at large $R$, confirming asymptotic scaling.}
\end{column}
\begin{column}{0.40\textwidth}
$\bar\Lambda$ diverges outside Class~I.
The natural generalizations:

\medskip
\textbf{Class II}\; ($\sigma^2\sim C_{II}\ln R$):
\[
  C_{II} = \lim_{R\to\infty}\frac{\sigma^2(R)}{\ln R}
\]
Period-Doubling:\; $C_{II} = 0.080$ (curve fit)

\bigskip
\textbf{Class III}\; ($\sigma^2\sim A_{III}\,R^{1-\alpha}$):
\[
  A_{III} = \lim_{R\to\infty}\frac{\sigma^2(R)}{R^{1-\alpha}}
\]
0222 chain ($\alpha=0.639$):\; $A_{III} = 0.209$ (curve fit)
\end{column}
\end{columns}
\end{frame}
\note{
Plateau method converges as $C+b/\ln R$ from above for Class~II and from below
for Class~III --- biased at finite $R$.
Curve fit to the full $\sigma^2(R)$ profile is unbiased: $C_{II}=0.080$, $A_{III}=0.209$.
Plateau cross-checks ($0.123$, $0.151$) systematically over/under-estimate.
}

\small
\begin{tabular}{lc}
\toprule
\textbf{Initial condition} & $\bar\Lambda$ \\
\midrule
Perturbed lattice  & $\approx 1/6$ \\

\end{tabular}

Both satisfy $S(k)=0$ for $|k|<K^*$.

\end{column}
\end{columns}
\end{frame}
\note{
Lattice: $\sigma^2(R)=\{2R\}(1-\{2R\})$ integrates to $1/12$ over each half-period,
so the running average is always $1/6$ at integer $R_\text{max}$.
Stealthy: starting from a perturbed lattice, the optimizer finds a near-lattice
ground state ($\bar\Lambda\approx1/6$); starting from random positions (collaborator
data), it finds the disordered phase ($\bar\Lambda=1/(\pi^2\chi)$).
Stealthy ground states are highly non-unique.
}

\Rightarrow\alpha=2$ exactly.
Rank-$\geq4$: $\det M=\lambda_1|\lambda_2|^2|\lambda_3|=1$; since $|\lambda_3|<1$
(Pisot), $|\lambda_2|>1/\sqrt{\lambda_1}$, so $\alpha<2$.
The complex pair hurts at higher rank: it pushes $\alpha$ below 2, not above it.
}

\begin{frame}[shrink=8]{Breaking the constraint: non-unimodular matrices}

If $|\det M|=1$ forces $\alpha\leq2$, what happens when we drop that?

\medskip
For any $2\times2$ Pisot matrix with all-real eigenvalues:
\[
  |\lambda_2| = \frac{|\det M|}{\lambda_1}
  \quad\Rightarrow\quad
  \boxed{\;\alpha = 3 - \frac{2\ln|\det M|}{\ln\lambda_1}\;}
\]

\begin{columns}[T]
\begin{column}{0.50\textwidth}
\small
\begin{tabular}{lcc}
\toprule
$|\det M|$ & Result & Example \\
\midrule
1 (unimodular) & $\alpha=3$ & all metallic means \\
$\in(1,\sqrt{\lambda_1})$ & $\alpha\in(2,3)$ & \textbf{fills the gap} \\
$\sqrt{\lambda_1}$ & $\alpha=2$ & (boundary) \\
\bottomrule
\end{tabular}

\vspace{0.5em}
28 non-unimodular $2\times2$ examples found;\\
all have $|\det|=2$, giving $\alpha\in(2.0,\,2.2)$.
\end{column}
\begin{column}{0.48\textwidth}
Smallest example ($|\det M|=2$):
\[
  M=\begin{pmatrix}0&1\\2&4\end{pmatrix}
\]
\textit{Rules:} $a\to b$,\; $b\to aabbbb$
$\lambda_1=2{+}\sqrt{6}\approx4.449$,\\
$|\det M|=2 < \sqrt{\lambda_1}\approx2.11$
\[
  \alpha = 3-\frac{2\ln2}{\ln(2+\sqrt{6})} = \mathbf{2.071}
\]
\end{column}
\end{columns}
\end{frame}
\note{
Formula $\alpha=3-2\ln|\det M|/\ln\lambda_1$ unifies all $2\times2$ cases.
For $\det=1$: recovers $\alpha=3$. Gap condition: $|\det M|\in(1,\sqrt{\lambda_1})$.
For $\lambda_1=4.449$: $\sqrt{\lambda_1}\approx2.109$, so $|\det|=2$ just fits.
To get $\alpha$ closer to 3, need $|\det M|$ close to 1 (hard with integers).
}

\begin{frame}[shrink=7]{Summary}

\begin{columns}[T]
\begin{column}{0.49\textwidth}
\textbf{From the assigned questions:}
\begin{itemize}
  \item BT: spreadability fails (dense Bragg + large $|\lambda_2|$);
    use $\bar\Lambda=0.377$
  \item $S(k)$ fitting unreliable; $\bar\Lambda$ stable from $N\sim1000$
  \item $2{<}\alpha{<}3$ empty for all $3\times3$ unimodular matrices;
    new $\alpha{=}2$ chain ($\bar\Lambda{=}0.275$) as byproduct
  \item $C_{II}{=}0.080$ (Period-Doubling),
    $A_{III}{=}0.209$ (0222 chain)
  \item Lattice $\bar\Lambda{=}1/6$ exact at all $N$;
    stealthy has ordered and disordered phases
  \item Stealthy $\alpha$ not well-defined; $\bar\Lambda$
    comparisons valid across densities
\end{itemize}
\end{column}
\begin{column}{0.49\textwidth}
\textbf{Extensions:}
\begin{itemize}
  \item[$\star$] No unimodular matrix (any rank) fills $2{<}\alpha{<}3$
  \item[$\star$] General formula $\alpha{=}3{-}2\ln|\det M|/\ln\lambda_1$
    for $2\times2$ Pisot matrices
  \item[$\star$] First $\alpha{=}2.071$ chain
    ($\bar\Lambda{\approx}0.650$, rough est.)
  \item[$\star$] Silver $\bar\Lambda{=}1/4$ confirmed at $0.3\sigma$
  \item[$\star$] Metallic-mean limit $=1/3$ (URL cloaked)
\end{itemize}

\vspace{0.5em}
\textbf{Open questions:}
\begin{itemize}
  \item Analytical proof of $\bar\Lambda(\text{Silver})=1/4$?
  \item Precise $\bar\Lambda$ for the $\alpha=2.071$ chain?
  \item Closed form for $\bar\Lambda(\mu_n)$?
  \item Can we find $\alpha$ closer to 3?
\end{itemize}
\end{column}
\end{columns}
\end{frame}
\note{
The non-unimodular result is the main new finding: the $\alpha\in(2,3)$ gap CAN
be filled, but only by breaking the unimodular constraint.
Silver$=1/4$ would be the first metallic-mean chain with a rational $\bar\Lambda$.
The limit $1/3=$ URL cloaked is structurally satisfying: as the chain approaches
maximum Class~I disorder, it converges to the disordered limit.
}
