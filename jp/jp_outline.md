# JP Outline

## 1. Introduction
- Frame the paper around the central question:
  how to classify and compare one-dimensional hyperuniform point patterns, especially substitution tilings, beyond the coarse three-class scheme.
- Motivate the need for both a classification metric and a ranking metric.
- Position the project relative to the current literature and the present state of the project.

## 2. Theory
- Introduce the two-axis framework:
  `\alpha` determines asymptotic hyperuniformity class.
- Introduce the second axis:
  `\bar{\Lambda}` ranks residual disorder within Class I.
- Present the substitution-matrix / eigenvalue framework for one-dimensional tilings.

## 3. Methods
- Pattern generation.
- Number variance and extraction of `\bar{\Lambda}`.
- Structure factor / integrated intensity methods for extracting `\alpha`.
- Spreadability as an auxiliary or diagnostic probe rather than a coequal primary metric.

## 4. Structural Constraints on `\alpha`
- Metallic means as the rank-2 unimodular case with `\alpha = 3`.
- Bombieri-Taylor as the `1 < \alpha < 2` example.
- Higher-rank / unimodular bounds on accessible `\alpha`.
- Non-unimodular families accessing `2 < \alpha < 3`.

## 5. Ranking Class I by `\bar{\Lambda}`
- Integer lattice as the minimum benchmark.
- Metallic means.
- URL.
- Bombieri-Taylor.
- Stealthy patterns.
- Emphasize that `\alpha` alone does not determine residual disorder.

## 6. Comparative Case Studies
- URL as a validation and comparison family.
- Stealthy patterns as a contrasting hyperuniform mechanism.
- Spreadability as a supporting measurement tool and its limitations.
- Additional comparative examples can remain here for now as the structure evolves.

## 7. Conclusion
- Return to the main thesis:
  one-dimensional hyperuniformity needs both a classification axis and a ranking axis.
- Summarize how substitution-matrix algebra constrains `\alpha`.
- Summarize how structural disorder is reflected in `\bar{\Lambda}`.
- End with open problems and unresolved interpretive questions.
